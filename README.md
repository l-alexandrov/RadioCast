<h1 align="center">Radio Cast</h1>


## Installation

1. `cp .env.example .env`

2. :wrench: Configure `.env` file

3. `php artisan key:generate`

4. `php artisan migrate`

5. `cp radio.example.xml ./storage/app/public/radio.xml`
